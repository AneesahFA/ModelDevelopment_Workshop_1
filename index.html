Machine Learning Algorithms — Visual Framework

A quick, visual guide to 10 widely used algorithms, organized by learning type with domains, use cases, and plain-English explanations.

Decision Tree • Supervised

If/then splits create a tree of decisions.

Tabular DataExplainable AI
  • Use cases: credit approval, churn prediction, medical triage.
  • How it works: recursively splits features at thresholds that best separate labels; leaves = final classes.

Random Forest • Supervised

Many trees vote to reduce overfitting.

Tabular DataRobust
  • Use cases: risk scoring, price prediction, feature importance.
  • How it works: trains trees on bootstrapped samples & random features; averages votes/predictions.

Support Vector Machine (SVM) • Supervised

Finds the maximum-margin decision boundary.

TabularNLP (kernels)Vision (HOG + SVM)
  • Use cases: text/category classification, anomaly detection (one-class SVM).
  • How it works: uses kernels to separate classes with the widest margin; support vectors define the boundary.

Logistic Regression • Supervised

Simple, strong baseline for probabilities.

TabularNLP (bag-of-words)
  • Use cases: spam vs. ham, fraud yes/no, conversion likelihood.
  • How it works: linear model with a sigmoid links features to class probabilities.

K-Means Clustering • Unsupervised

Groups points by nearest centroid.

TabularVision (color clusters)
  • Use cases: customer segmentation, image color quantization, document grouping.
  • How it works: iteratively assigns points to the closest mean and updates centroids until stable.

Principal Component Analysis (PCA) • Unsupervised

Reduces dimensions while preserving variance.

PreprocessingVisualization
  • Use cases: noise reduction, feature compression, 2D plotting of high-dim data.
  • How it works: rotates data to new orthogonal axes (components) ordered by explained variance.

Convolutional Neural Network (CNN) • Deep Learning

Extracts spatial patterns from images.

Computer VisionEdge/Texture
  • Use cases: image classification, object detection, medical imaging.
  • How it works: convolutions + pooling capture local features; deeper layers build complex concepts.

RNN / LSTM • Deep Learning

Remembers sequence context over time.

NLPTime Series
  • Use cases: language modeling, forecasting, speech recognition.
  • How it works: recurrent connections pass hidden state; LSTM gates control memory/forgetting.

Generative Adversarial Networks (GANs) • Generative AI

Two nets compete: generator vs. discriminator.

VisionSynthetic Data
  • Use cases: image synthesis, style transfer, data augmentation.
  • How it works: generator creates samples; discriminator critiques; both improve through adversarial training.

Transformers (e.g., BERT, GPT) • Deep / NLP

Self-attention models long-range dependencies.

NLPMultimodal
  • Use cases: translation, Q&A, summarization, code & text generation.
  • How it works: attention weights relate every token to every other, enabling parallel training and context understanding.

How to read this page: Each card lists the algorithm type (learning paradigm), common application domains, real-world use cases, and a short explanation. Together they form a concise infographic-style portfolio artifact.
